Bienvenido a mi portal personal. Este proyecto es más que un simple portafolio; es un ecosistema diseñado para mostrar mis habilidades técnicas, proyectos y pasión por la inteligencia artificial y el desarrollo web moderno.

## 🚀 Características Principales

- **🤖 IAnko Chatbot**: Un asistente inteligente integrado entrenado con mi perfil profesional, capaz de responder preguntas sobre mi experiencia y habilidades en tiempo real.
- **💻 Terminal Interactiva**: Una interfaz de línea de comandos (CLI) funcional dentro del navegador para los amantes del desarrollo.
- **🌐 Globo 3D Interactivo**: Visualización de conectividad y geolocalización utilizando `globe.gl`.
- **⚡ Rendimiento Extremo**: Construido con Next.js 15 (App Router) y Tailwind CSS 4 para una experiencia fluida y rápida.
- **📱 Responsivo y Premium**: Diseño orientado a la excelencia visual con modo oscuro profundo y animaciones sutiles.

## 🛠️ Stack Tecnológico

- **Frontend**: Next.js 15, React 19, Lucide React (Icons).
- **Estilos**: Tailwind CSS 4, CSS Moderno.
- **Componentes Especiales**: `globe.gl`, `sonner` (toasts).
- **Despliegue**: Cloudflare Pages / Wrangler.
- **IA**: Integración con modelos de lenguaje vía API de Cloudflare Workers (IAnko).

## 🛠️ Instalación y Desarrollo

Para ejecutar este proyecto localmente:

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/yankoacuna/yanko-dev-portal.git
   ```

2. **Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Ejecutar servidor de desarrollo**:
   ```bash
   npm run dev
   ```

4. **Abrir en el navegador**:
   Visita `http://localhost:3000` para ver el resultado.

## 🌍 Despliegue

Este proyecto está optimizado para **Cloudflare Pages**. El flujo de despliegue automatizado compila la aplicación usando `@cloudflare/next-on-pages`.

Para desplegar manualmente:
```bash
npm run deploy
```
